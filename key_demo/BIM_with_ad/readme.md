# [小红砖](www.bos.xyz) 示例库-BIM与广告 代码导航


BIM与广告，是将BIM与广告结合起来，改变了以往使用纸质宣传手册或仅使用平面进行宣传的方式，也可以将复杂的安装流程分步演示，通过动效的方式，直观立体地说明每一步是如何实现的。这是建筑信息模型在广告宣传行业的一大应用，改变了传统广告行业使用纸质宣传或录制视频的方式，让宣传与使用更加富有科技感。
