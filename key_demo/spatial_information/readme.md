# [小红砖](www.bos.xyz) 示例库-空间信息示例代码导航


BIM模型空间树的应用场景众多，可应用于建筑分层导览展示，如机场航站楼、高铁站、医院等，也可以应用与智能楼宇的监控与管理中，也可应用于建筑运维管理中。通过使用建筑空间树，可以清晰的展现建筑每一个层次的空间样貌，同时有又保持了良好的空间结构与层次性，从而避免了因楼层内部结构复杂而引起的导览不明确，无法立体展示的缺陷。
