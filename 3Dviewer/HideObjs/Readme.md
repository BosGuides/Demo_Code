# 文档说明

当前文档用于存放：

     示例库->三维示例->隐藏

中每个示例的代码。

网站访问地址为：
  http://www.boswinner.com.cn/group/list




文件与示例库模型对应关系如下：

|     文件名称      |  对应示例   |
| :-----------: | :-----: |
| ComponentHide |  构件隐藏   |
|   TypeHide    |  构件类隐藏  |
|   HouseHide   | 住房类-隐藏  |
|  OfficeHide   | 办公区类-隐藏 |
|  CompanyHide  | 厂房类-隐藏  |
