# [小红砖](www.bos.xyz) 示例库-剖切 代码导航


剖切示例，可以从六个面对整个模型进行剖切，方便查看建筑在不同维度的形态，还可以对任意选中三维构件或构件组隔离后进行单独剖切，方便在住房类、办公区、厂房类等复杂的模型中清晰便捷地查看各维度的细节展示。

### 在当前代码库中，提供了可以供开发者学习和使用的示例代码。

#### 对应可视化参考，可前往[小红砖示例库](https://www.bos.xyz/examples/)
