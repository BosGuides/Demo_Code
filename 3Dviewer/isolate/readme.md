# [小红砖](www.bos.xyz) 示例库-隔离 代码导航


隔离示例，可以将模型中任意三维构件或构件组隔离，使其单独显示，模型中其余构件均隐藏。方便在住房类、办公区、厂房类等构件较多的模型中快速准确地查看所需构件，减少干扰因素。

### 在当前代码库中，提供了可以供开发者学习和使用的示例代码。

#### 对应可视化参考，可前往[小红砖示例库](https://www.bos.xyz/examples/)
